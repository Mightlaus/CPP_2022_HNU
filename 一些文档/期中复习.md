# 练习六 指针

sort()数组排序时用法：

sort(*first, *last) ： first与last均为地址；排序范围**包含first不包含last[first, last)**

用.begin() .end()时不需要考虑这个问题

```cpp
int main() {
    int m=3, n=4;

    int a[7] = {7,6,5,4,3,2,1};
	int b[n] = {...}
    sort(&a[0], &a[6]); // a = {2,3,4,5,6,1} 不包含a[6](last)
    sort(&b[0], &b[n]); // 对b数组全排序
    sort(b, b+n); // 同上
    cout<<0;

}
```
